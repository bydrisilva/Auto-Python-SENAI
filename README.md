# AUTO PYTHON.
## Sistema de controle de veículos. 
Um sistema robusto e completo que gerencia veículos em uma concessionária. 

## Funcionalidades:
### Cadastro:
Vai realizar o cadastro de veículos. Usando o Modelo, Cor, Quilometragem, Ano e Marca. Antes do cadastro o sistema irá validar os dados. 

### Atualização:
Irá atualizar os dados de um veículo já existente na base de dados, antes de alterar, o sistema valida os dados.

### Remoção: 
Valida se o veículo existe antes de tentar remover.
### Leitura:
Exibe todos os veículos disponíveis no pátio da concessionária e permite buscar modelos específicos para melhor visualização.
